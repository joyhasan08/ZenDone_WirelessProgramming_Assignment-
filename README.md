# ZenDone AI To-Do

ZenDone is a sophisticated, high-performance To-Do application designed for focus and productivity. It features local state management, elegant UI transitions, and intelligent task motivation powered by Google's Gemini AI.

## ðŸš€ Features

- **AI Motivation**: Dynamic, context-aware motivational tips generated by Gemini based on your progress.
- **Glassmorphic UI**: A modern, clean aesthetic using Tailwind CSS and backdrop blurs.
- **Persistent Storage**: Your tasks are saved locally in your browser using `LocalStorage`.
- **Filtering**: Quickly toggle between All, Pending, and Completed tasks via a sticky bottom navigation.
- **Responsive Design**: Fully optimized for mobile and desktop screens.

## ðŸ’» Local Installation & Setup

To run this project on your local machine, follow these steps:

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd zendone-todo
```

### 2. Install Dependencies

If you are setting this up as a standard Node project, run:

```bash
npm i
```

_Note: You may also need to install dev dependencies like `typescript`, `@types/react`, and `tailwindcss` if setting up from scratch._

You can obtain an API key from the [Google AI Studio](https://aistudio.google.com/).

### 5. Running the App

For a standard Vite/React setup:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app in your browser.
